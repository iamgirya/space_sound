# Sound Space

### Размещён: https://space-sound.onrender.com/

Пространственный аудио-плейграунд, работающий целиком в браузере — без установки, без сервера. Создавался полезным для настольно ролевых игр.

Вдохновлено [TAT by Paradiso](https://www.paradiso.zone/ooo-tat/).

> Вайбкодинг с помощью Claude.

---

## Что это такое?

Sound Space — это одностраничное веб-приложение для создания интерактивных пространственных аудио-сцен. Вы размещаете звуковые зоны на двумерном холсте, располагаете слушателя, и звуки включаются, когда слушатель входит в зону — с громкостью, зависящей от близости к центру зоны. Добавляйте изображения и рисунки от руки, чтобы создать визуальный фон. Всё автоматически сохраняется в браузере и может быть экспортировано одним портативным файлом.

---

## Возможности

- **Пространственные аудио-зоны** — круги и полигоны, звуки плавно появляются и исчезают при входе/выходе
- **Громкость по близости** — звук усиливается к центру зоны
- **Глобальные и постоянные звуки** — воспроизведение независимо от положения слушателя
- **Таймеры звуков** — запуск звука через N секунд с опцией повтора
- **Изображения на холсте** — перемещаемые, масштабируемые, закрепляемые
- **Рисование от руки** — маркер с выбором цвета и толщины
- **Несколько страниц** — независимые сцены в одной сессии
- **Панорамирование и масштабирование щипком** — навигация на десктопе и мобильном
- **Автосохранение** — состояние сохраняется в IndexedDB между перезагрузками
- **Экспорт / Импорт** — полное сохранение в виде одного самодостаточного JSON-файла

---

## Где открывается

Sound Space — это один `.html`-файл. Откройте его в любом современном браузере:

- **Десктоп** — Chrome, Firefox, Safari, Edge
- **Мобильный** — iOS Safari, Android Chrome
- **Сервер не нужен** — файл открывается напрямую (`file://`) или через любую статическую раздачу

Сенсорные жесты полностью поддерживаются: перетаскивание для панорамирования, щипок для масштабирования, двойной тап для телепортации слушателя или закрепления изображения.

## Подробный обзор возможностей

### Слушатель

Слушатель (`👂`) — это «ухо» на холсте. Перетаскивайте его по карте, чтобы услышать, какие звуки активны в данной точке. Двойной тап (или двойной клик) на пустом месте мгновенно телепортирует слушателя.

### Звуковые зоны

- **Круговые зоны** (`⊕ Круг`) — рисуются перетаскиванием. Радиус задаёт границу звука.
- **Полигональные зоны** (`⬡ Полигон`) — кликайте для добавления вершин, двойной клик или клик по первой точке закрывает контур. `Esc` для отмены.
- У каждой зоны есть название, цвет, назначенный звук и регулятор громкости.
- **Громкость по близости** — при включении звук наиболее громкий в центре зоны и затихает к краю.
- **Не останавливать** — при выходе слушателя звук затихает, но продолжает воспроизводиться с той же позиции. При возвращении громкость восстанавливается.
- **Глобальное воспроизведение** — звук зоны играет везде на странице с фиксированной громкостью, независимо от положения слушателя.
- **Ручное управление** — у каждой зоны в списке есть кнопка воспроизведения/остановки и ползунок перемотки.
- Активные зоны анимируются пульсацией.

### Звуки

- Загружайте любое количество аудиофайлов через кнопку загрузки.
- Прослушивайте звуки перед назначением.
- **Глобальные звуки** — включите постоянное воспроизведение звука с регулятором громкости.
- **Таймеры** — задайте обратный отсчёт, после которого звук становится глобальным. Поддерживается повтор (🔁): после окончания звука таймер перезапускается с теми же задержкой и громкостью.

### Изображения

- Загружайте изображения и размещайте их на холсте.
- Свободно перемещайте и масштабируйте с помощью четырёх угловых маркеров.
- **Закрепление** — двойной тап (или двойной клик) на изображении фиксирует его на месте. Закреплённые изображения выделяются оранжевой рамкой и отмечаются значком 🔒 в панели. Повторный двойной тап — открепить.

### Рисование

- **Маркер** (`✏`) — рисование от руки мышью или пальцем.
- Выбор из 8 заготовленных цветов или ввод произвольного hex-кода.
- Три варианта толщины линий.
- Каждый штрих — отдельный объект: в режиме перемещения его можно выделить и перетащить.
- Удалите выделенный штрих клавишей `Delete` или `Backspace`.

### Навигация по холсту

- **Панорамирование** — перетаскивание на пустом месте (мышь или один палец).
- **Масштабирование** — колесо мыши на десктопе; щипок двумя пальцами на мобильном.
- **Сброс вида** — кнопка `⌂ Центр` сбрасывает панорамирование и масштаб к исходному состоянию.

### Страницы

- Создавайте несколько независимых сцен (вкладки над холстом).
- У каждой страницы свои зоны, изображения, рисунки и положение слушателя.
- Библиотека звуков общая для всех страниц.
- Переименование — двойной клик по вкладке. Удаление — кнопка `×` (минимум одна страница).

### Сохранение и экспорт

- **Автосохранение** — каждое изменение с задержкой записывается в IndexedDB. Всё восстанавливается после перезагрузки страницы.
- **Экспорт (`⬇`)** — скачивает файл `soundspace-save.json`, содержащий все аудио- и графические данные в base64. Полностью самодостаточный.
- **Импорт (`⬆`)** — загружает файл сохранения и полностью восстанавливает сессию.

## Технические особенности

- Один `.html`-файл — никакой сборки, npm или фреймворков
- Всё хранилище через IndexedDB (аудио-блобы, блобы изображений, полное состояние приложения)
- Рендеринг холста с трансформацией pan/zoom (`ctx.translate` + `ctx.scale`)
- Pointer Events API для единой обработки мыши и касаний (`setPointerCapture` для надёжного перетаскивания)
- Масштабирование щипком через отслеживание двух указателей с математикой масштабирования относительно средней точки
